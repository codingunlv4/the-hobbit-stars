Misty Mountains background music
================================

Place your audio file in this folder with one of these names:

  misty_mountains.ogg   (recommended)
  misty_mountains.mp3

The game loops it during the character creator and battles.

Vocal or instrumental — both work
---------------------------------
The game plays whatever file you add. There is no restriction on vocals.

  - Vocal version: the sung "Far Over the Misty Mountains Cold" track
  - Instrumental version: a cover or soundtrack without singing

Either format (OGG or MP3) is fine. Instrumental tracks often loop more
smoothly during gameplay, but the full vocal version works if you prefer it.

Licensing
---------
Use a copy you own or are licensed to use (for example, from a
soundtrack you purchased). The game does not include copyrighted
music by default.

After adding the file, reopen the project in Godot or press F5 to play.
